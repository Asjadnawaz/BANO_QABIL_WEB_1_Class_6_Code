# 📊 HTML Table – Student Details

This project is a basic HTML practice task where teacher tells how to create tables in HTML.

In this session, the teacher introduced how to structure data using the `<table>` tag and its related elements.

## 📚 What I Learned
- Creating tables using `<table>` tag
- Using table rows `<tr>`
- Using table headers `<th>` and data cells `<td>`
- Merging columns using `colspan`
- Basic table attributes like `border`, `cellpadding`, and `cellspacing`

## 🎯 Practice Work
- Created a student details table
- Displayed name, email, password, and roll number in structured format
- Used `colspan` for table heading

## 🚀 Goal
To understand how to organize and display data in a tabular format using HTML.
